# TaskMate

TaskMate is a task management web application built using ASP.NET Core MVC.

The project was developed end-to-end, starting from database design, through backend logic, and up to the frontend layer, following the MVC pattern.

## Features
- User authentication and authorization using ASP.NET Core Identity
- Task management (create, update, delete, list)
- Role-based access
- Entity Framework Core for database operations
- SQL Server database

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server
- Bootstrap
